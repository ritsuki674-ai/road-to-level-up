# Road to Level Up

英語スピーキング、ギター、イラスト、3Dオブジェクト制作を、EXP・レベル・クエスト形式で可視化する個人用MVPアプリです。

## 起動

```bash
python3 -m http.server 8123
```

Macのブラウザでは `http://localhost:8123` を開きます。`8123` が使われている場合は別ポートを指定してください。

```bash
python3 -m http.server 8000
```

同じWi-Fi内のスマホから試す場合は、MacのIPアドレスを使って `http://MacのIPアドレス:8123/` を開きます。

## PWA・オフライン利用

このアプリはPWAとして使えるように、`manifest.webmanifest` と `service-worker.js` を含めています。

- ホーム画面に追加すると、アプリのように起動できます。
- 一度読み込んだ後は、HTML/CSS/JSなどの基本ファイルがキャッシュされます。
- Mac側でファイルを変更して新しい版が見つかると、画面下に更新通知が表示されます。
- 「更新」を押すと、新しいキャッシュへ切り替えて再読み込みします。

アプリを更新したときは、`index.html` の `styles.css?v=...` と `app.js?v=...`、`app.js` の `APP_VERSION`、`service-worker.js` の `CACHE_VERSION` とキャッシュ対象URLを同じ新バージョンへ上げると、スマホ側に更新が届きやすくなります。

注意: service worker はブラウザの仕様上、`localhost` またはHTTPSのような安全な接続で動きます。スマホからMacの `http://192.168...` にアクセスする場合、ブラウザによってはPWAのオフライン機能が有効になりません。スマホ単体で安定して使う場合は、GitHub PagesやNetlifyなどのHTTPS配信に置くのが確実です。

データは端末ごとのブラウザ内 `localStorage` に保存されます。Macとスマホ間で同期したい場合は、設定画面のJSONエクスポート・インポートを使います。

## MVP機能

- 朝の入力: 睡眠時間、起床時間、今日使える時間
- ルールベースの今日のクエスト提案
- クエスト完了チェック、EXP加算、レベルアップ表示
- スキル一覧、クエスト手動登録、履歴表示
- 英語ログ入力とChatGPT添削用プロンプト生成
- イラスト・3D用画像ログ
- マネージャー画像枠と差し替え
- JSONエクスポート・インポート
- 端末内 `localStorage` 保存

## オフライン方針

外部AI APIは使用しません。英語添削はアプリ内では行わず、ChatGPTなどに貼るためのプロンプト生成だけを行います。

画像ログはMVPでは `localStorage` にBase64保存しています。将来的にIndexedDBへ差し替えやすいよう、画像ログデータは独立した配列で管理しています。
